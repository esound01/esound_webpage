# eSound Imaging Website

Professional portable ultrasound and diagnostic imaging services website for eSound Imaging.

## Features

- **Bilingual Support**: Full English and Spanish language switching
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Professional design with smooth animations
- **Services Showcase**: Display of all diagnostic services offered
- **Contact Form**: Easy-to-use contact form for service requests
- **Testimonials**: Client testimonials section
- **About Section**: Company information and mission statement

## Project Structure

```
esound_webpage/
├── index.html          # Main HTML file

├── js/
```

## Technologies Used

- HTML5
- CSS3 (Responsive Design)

- Google Fonts (Poppins, Inter)

## Language Support

The website supports both English and Spanish with a language switcher in the top-right corner. Language preference is saved in localStorage.

## Deployment

This website is designed to be deployed on Cloudflare Pages or any static hosting service.

### Cloudflare Pages Deployment

1. Push code to GitHub repository: `esound01/esound_webpage`
2. Connect GitHub to Cloudflare Pages
3. Configure custom domain: `esoundimaging.com`
4. Deploy automatically on push

## Local Development

To view the website locally:

1. Clone the repository
2. Open `index.html` in a web browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```
